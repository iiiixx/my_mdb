# Система рекомендаций фильмов

Программный комплекс для формирования персонализированных рекомендаций фильмов на основе пользовательских рейтингов.
Рекомендации вычисляются с использованием модели **матричной факторизации (SVD)**, которая позволяет предсказывать предпочтения пользователей на основе их прошлых оценок.

Система реализована как набор сервисов:

* **Go backend** — обработка HTTP-запросов, управление данными и формирование ответов API
* **Python сервис рекомендаций** — вычисление рекомендаций на основе обученной модели
* **PostgreSQL** — хранение фильмов, рейтингов и статистики

---

## Возможности системы

* персонализированные рекомендации фильмов для пользователя
* поиск **похожих фильмов**
* выставление и обновление **пользовательских рейтингов**
* отображение **популярных фильмов (топ)**
* подборки фильмов **по жанрам**
* **динамический блок рекомендаций** с элементом случайности
* получение постеров и дополнительной информации о фильмах через **OMDb API**

---

## Архитектура

Система имеет микросервисную архитектуру.

```
Пользователь
      ↓
Go Backend (HTTP API)
      ↓
Service layer
   ↓        ↓
PostgreSQL   gRPC
               ↓
      Python Recommender
            (SVD)
```

Backend отвечает за обработку запросов и управление данными, а сервис рекомендаций выполняет вычисления модели.

---

## Алгоритм рекомендаций

Используется модель **SVD с учетом смещений пользователей и фильмов**.

Предсказанный рейтинг вычисляется как:

r̂_ui = μ + b_u + b_i + q_iᵀ p_u

где:

* μ — средний рейтинг
* b_u — смещение пользователя
* b_i — смещение фильма
* p_u — латентный вектор пользователя
* q_i — латентный вектор фильма

Рекомендации формируются на основе фильмов с наибольшим предсказанным рейтингом.

---

## Основные технологии

Backend:

* Go
* PostgreSQL
* gRPC

Recommendation service:

* Python
* NumPy
* Surprise (SVD)

Дополнительно:

* OMDb API
* HTML templates + CSS + JS
